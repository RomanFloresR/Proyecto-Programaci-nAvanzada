Titulo: V.A.S.L (Vehiculo Automatico Seguidor de Linea)

Objetivo General:
* Crear un robot automatico capaz de seguir una linea a traves de sensores infrarojos ademas es posible manejarse a traves de un dispositivo de control remoto

Objetivos Especificos:
* Aprender el uso de una placa FPGA
* Conocer la interfaz de dispositivos de control remoto con otros dispositivos
* Aprovechar la facilidad de python de hacer interfaz con otros sistemas
