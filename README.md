# Proyecto V.A.S.L (Vehiculo Automatico Seguidor de Linea) #

Repositorio con la información para el proyecto de progración avanzada V.A.S.L

Objetivos general:

- Crear un robot automatico capaz de seguir una linea a traves de sensores infrerojos ademas de ser posible de manejarse a traves de un dispositivo de control remoto

Objetivos especificos:

- Aprender el uso de una placa FPGA
- Conocer la interfaz de dispositivos de control remoto con otros dispositivos
- Aprovechar la facil capacidad de python de hacer interfaz con otros sistemas
